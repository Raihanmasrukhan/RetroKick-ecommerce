# RetroKick-ecommerce
Toko Sepatu Gaul

Welcome to RetroKick, an elegant, minimalist, and "retro gaul" e-commerce application built with Laravel and Vanilla CSS.

## Features
- Fullstack Laravel application.
- SQLite / MySQL support.
- Fully customized Vanilla CSS styling for a unique 90s aesthetic.
- Authentication (Login/Register) seamlessly integrated with the design.
- Persistent Shopping Cart (saved to the database for authenticated users).
- Retro Checkout Pop-up Modal.

## Installation
1. Clone this repository.
2. Run `composer install`.
3. Run `npm install` and `npm run build`.
4. Copy `.env.example` to `.env` and configure your database.
5. Run `php artisan key:generate`.
6. Run `php artisan migrate --seed` to generate the tables and dummy products.
7. Run `php artisan serve` to start the local development server.
